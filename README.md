LinguaLab — это веб-приложение для изучения иностранных языков, созданное с помощью Firebase Studio. Его главная особенность — персональный подход к обучению, который обеспечивается благодаря использованию генеративного искусственного интеллекта (Genkit и Google Gemini).

## Основные возможности

### Что уже реализовано в приложении:

1.  **Регистрация и персонализация (Onboarding):**
    *   При первом входе пользователь проходит опрос: указывает имя, выбирает язык интерфейса и язык для изучения, а также описывает свои цели.
    *   На основе этих данных **AI генерирует персональный учебный план** (роадмап) от уровня A1 до C2, который становится основой для дальнейшего обучения.

2.  **Главная панель (Dashboard):**
    *   **Учебный план:** Отображается полный план обучения в виде интерактивного аккордеона. Пользователь может разворачивать уроки, видеть темы и отмечать их как выполненные. Ссылки на темы (например, "Грамматика: ...", "Лексика: ...") автоматически ведут в соответствующие учебные модули.
    *   **AI-рекомендации:** Есть кнопка, которая анализирует прогресс пользователя и предлагает следующий наиболее подходящий урок для изучения.
    *   **Отслеживание цели и прогресса:** Визуально отображается прогресс в выполнении учебного плана.
    *   **Советы от AI-репетитора:** Периодически генерируются короткие полезные советы по изучению языка.
    *   **Навигация по модулям:** Есть карточки для быстрого перехода ко всем основным учебным разделам.

3.  **Учебные модули (частично реализованы):**
    *   **Грамматика, Лексика, Аудирование, Чтение, Письмо, Говорение:** Для каждого из этих разделов есть отдельная страница. При переходе из учебного плана тема урока автоматически передаётся в модуль.
    *   Внутри каждого модуля **AI генерирует уникальные задания "на лету"**: тексты для чтения, диалоги для аудирования, темы для говорения, списки слов и т.д.
    *   **Практика слов и Повторение:** Модули для закрепления лексики с помощью упражнений.

4.  **Профиль и Настройки:**
    *   **Прогресс и Достижения:** Страницы для отслеживания очков опыта (XP), серий (streak) и заработанных значков (элементы геймификации).
    *   **Архив ошибок:** Все ошибки, которые пользователь совершает в практических заданиях, сохраняются. Их можно просмотреть, чтобы понять свои слабые места.
    *   **Настройки:** Можно изменить свои цели, интересы и тему оформления приложения (светлая/тёмная). Также есть возможность сбросить весь прогресс и начать заново.

### Что ещё в разработке или не реализовано:

*   **Глубокая интеграция некоторых модулей:** Хотя большинство модулей генерируют контент, взаимодействие между ними и сохранение прогресса по *каждому отдельному заданию* может быть неполным. Например, модуль фонетики пока может использовать статические (заранее заданные) задания, а не генерируемые AI.
*   **Детальная аналитика:** Страница "Прогресс" пока показывает только общую статистику (XP, значки). Углублённая аналитика, вроде "дерева навыков", ещё не реализована.
*   **Сложная геймификация:** Система значков (баджей) реализована, но можно добавить более сложные механики, например, ежедневные задания или награды за выполнение определённых типов уроков.

В целом, у вас есть очень мощная основа: ядро приложения с AI-генерацией персонализированного контента и основные экраны полностью функционируют.
